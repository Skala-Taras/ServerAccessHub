# ServerAccessHub

Personal cloud storage with web terminal in Docker.

## Quick Start (Local)

```bash
docker-compose up --build -d
```

Open: https://localhost:8080

## Deploy to Server

See [DEPLOY.md](DEPLOY.md) for Ubuntu server deployment with Ansible.

## Features

- File browser (upload, download, delete)
- Web terminal (bash)
- PDF, image, video preview
- Download folders as ZIP
